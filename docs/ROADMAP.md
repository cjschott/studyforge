# Roadmap

## v0.3 Backend/accounts/db

FastAPI backend, SQLite schema, local auth, admin users, import/export, optional frontend backend mode.

## v0.4 Learning engine/spaced repetition

Improve mastery modeling, flashcard scheduling, weak-topic targeting, and readiness scoring.

## v0.5 Course Builder

Build guided course setup, source ingestion, validation dashboards, and human review queues.

## v0.6 AI Tutor

Add local/self-hosted-friendly tutor workflows with source-cited explanations and guardrails.

## v1.0 multi-course platform

Polish multi-course administration, student/instructor workflows, deployment hardening, and backups.
