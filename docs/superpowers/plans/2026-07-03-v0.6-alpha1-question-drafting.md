# StudyForge v0.6.0-alpha.1 Question Drafting Plan

## Objective

Add a non-AI question drafting foundation that creates review-gated draft questions from source chunks and concepts, preserves lineage, surfaces validation warnings, and requires explicit publishing before drafts enter the live question bank.

## Backend

- [x] Add `question_drafts` and `question_draft_lineage` models.
- [x] Add authenticated draft CRUD, review, verify, reject, and publish endpoints.
- [x] Add rule-based source-material drafting for practice-question-like chunks.
- [x] Add concept drafting with reviewable placeholder choices.
- [x] Add Course Builder drafting endpoint for selected sources.
- [x] Add validation warnings for missing answer, explanation, lineage, wrong-answer explanations, unverified sources, and unresolved conflicts.
- [x] Implement idempotent publishing through `published_question_id`.

## Frontend

- [x] Add Question Drafts sidebar screen.
- [x] Add draft list filters and detail editor.
- [x] Add lineage and warning rendering.
- [x] Add review, verify, reject, and publish controls.
- [x] Add draft controls to Source Material, Concept Detail, Course Builder, and Admin review queue.

## Verification

- [x] Add backend tests for manual draft creation, source/concept/Course Builder drafting, warnings, review statuses, rejected visibility, and publish idempotency.
- [x] Add frontend tests for draft list/detail/filter/publish helpers and Course Builder draft counts.
- [x] Run full backend and frontend verification.
