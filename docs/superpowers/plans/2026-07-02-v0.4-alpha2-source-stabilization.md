# StudyForge v0.4 Alpha.2 Source Stabilization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stabilize the v0.4 Source Library foundation and let Course Builder select source materials without adding AI generation.

**Architecture:** Keep source storage/extraction in the existing FastAPI Source Library router and expose safe material summaries for frontend use. Add a dedicated Course Builder frontend module that reads Source Library APIs, lets users select materials, and builds a local source context preview without persisting generation jobs.

**Tech Stack:** FastAPI, SQLAlchemy, SQLite, vanilla JS modules, Node `node:test`, pytest.

---
